# DSA-in-Java 🚀

A collection of my Data Structures and Algorithms (DSA) practice problems and solutions, written in Java. This repository is a running log of my coding practice — organized topic-wise, updated regularly as I solve new problems and revisit core concepts.

## 📌 About

This repo is not a tutorial or a finished project — it's my personal DSA journey, made public to track consistency and progress over time. Expect daily/regular commits as I work through topics from basics to advanced.

## 📂 Folder Structure

Each topic has its own folder containing relevant `.java` files:

```
DSA-in-Java/
├── Array/
├── String/
├── LinkedList/
├── Stack/
├── Recursion/
├── Backtracking/
├── Hashing/
├── Divide-Conquer/
├── Sorting/
├── Bit-Manipulation/
├── Conditional-Statements/
├── Loops/
├── Functions/
├── Pattern/
└── ...more added as I progress
```

Each `.java` file is self-contained and can be run independently to see the output for that specific problem.
